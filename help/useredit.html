User editable files

There are 3 files the user may wish to modify to suit their preferences. They are:
nvlocations.txt Location file - list of observing locations around the world, used by the Set location window
nvdeepsky.txt Deep sky object file - list of all deep sky objects (nebulas, galaxies, star clusters, etc) displayed by the program
nvstarnames.txt Star name file - list of all star names displayed by the program

For example, the user may wish to modify nvstarnames.txt to add or alter the star names used by Night Vision.

Use the Write user files window (via the "File" pull down menu) to create these files. Night Vision will look for each file when it starts up. If found, the file's data will be used in place of the program's internal data. The location of the files is described here.

Directions on how to modify are at the top of each file. It is important to follow these directions carefully. (Any errors found by the program on start up will be listed in the Messages window.) Use a standard text editor. (E.g. Windows users can use "Notepad".) Do not use a word processor unless it can save to a standard ASCII file. Also do not save with tabs.
