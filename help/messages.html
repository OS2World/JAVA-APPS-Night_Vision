Messages window

This window displays messages generated by Night Vision. Generally these are errors found while reading user editable files during startup. For example, if a line in the deep sky object file, nvdeepsky.txt, has an error, that error will be listed in this window, hopefully with an explanation sufficient enough to tell how to correct it.

The Messages window is invoked by selecting "Show messages" from the "File" pull down menu. If there are no messages to show, this selection will grayed out.
