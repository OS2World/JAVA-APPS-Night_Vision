Set fonts window

Use this window to set the fonts for the various objects in the main star window. These fonts are also used for printing star charts. (Bayer and Flamsteed star labels use a plain font that has the same size as that used for star names.) Select the object in the top listbox, then modify its font below. ("Horizon markers" refer to compass directions along the horizon ("N", "NE", "E", ...).)

Pressing "OK" or "Apply" accepts new values.
Pressing "Cancel" rejects any changes.
