The Shift and Ctrl keys alter operation of the mouse. If either are pressed before pressing mouse button 1 and dragging the mouse, then a zoom rectangle is drawn rather than moving the view. When the mouse button is released the center of the rectangle is moved to the center of the window at a zoom level based on the area of the rectangle. If mouse button 1 is simply clicked with the Shift or Ctrl key down then the point clicked on will move to the center of the window.

The following keys are active in secondary windows and the help window:
Key Function
Tab Navigate forward through controls
Shift-Tab Navigate backward through controls
Space Activate button or select/deselect a checkbox
Enter Activate default button (in most cases)
Down arrow Move down in list
Up arrow Move up in list
Alt-Down arrow Show/retract list in drop-down listbox (combo-box)
Alt-Up arrow Show/retract list in drop-down listbox (combo-box)
F1 Show help
Ctrl-Home Go to top of help page
Ctrl-End Go to bottom of help page
Escape Cancel/Close window

'CapsLock' is ignored.

Although a mouse facilitates most operations, nearly all functions can be done with the keyboard. A few notable exceptions:

  • Links on help pages can only be activated with a mouse.
  • Objects can only be identified with a mouse.
  • The View Rotation scrollbar can only be changed with a mouse.
