Find deep sky object window

Use this window to find a deep sky object (i.e. galaxy, nebula, ...).

The checkboxes can be selected/deselected to show only objects of particular types, and the text field can be utilized to look for a character string (e.g. to list only NGC objects, type in "ngc"; case is unimportant).

Note: The object may not be visible if it is too dim for current magnitude settings or if the showing of deep sky objects has been turned off.

Pressing "OK" or "Apply" locates the selected object.
Pressing "Cancel" dismisses the window.
