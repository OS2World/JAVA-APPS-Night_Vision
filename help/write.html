Write user files window

Use this window to write the following 3 files to your computer:
nvlocations.txt Location file - list of observing locations around the world, used by the Set location window
nvdeepsky.txt Deep sky object file - list of all deep sky objects (nebulas, galaxies, star clusters, etc) displayed by the program
nvstarnames.txt Star name file - list of all star names displayed by the program

These files can be altered by the user to change the data used by the program.

When Night Vision is starting, it will look for each file. If found, the file's data will be used in place of the program's internal data. See User editable files for additional information.

On non-Windows machines these files are written into the directory (folder) where Night Vision is installed. For Windows XP, Vista, 7, 8, and 10 these files are placed in a "Night Vision" folder within the user's "Documents" folder. Use Windows Explorer to locate the files. For XP, this location is typically: 

C:\Documents and Settings\<username>\My Documents\Night Vision
For Vista and 7, this location is typically: 
C:\Users\<username>\Documents\Night Vision
For Windows 10, this location is typically: 
C:\Users\<username>\Desktop\Documents\Night Vision
Within this location a double click with the left mouse button on the file icon will typically invoke the Notepad editor, which is ideal for modifying it. (Avoid word processors such as Wordpad and Word which will embed formatting codes that can not be interpreted.)
